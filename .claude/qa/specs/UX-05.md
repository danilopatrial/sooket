## What this tests
Verifies that key interactive elements support keyboard navigation: VarField autocomplete suggestions, node search overlay, and Debug panel preset saving all respond to standard keyboard shortcuts.

## Steps — VarField autocomplete keyboard navigation

1. Open a canvas node with a VarField (e.g. Template String node's template field or If node's Compare To)
2. Type `$` or `{{$node.` to trigger autocomplete suggestions
3. Press **↓ (ArrowDown)** — highlight advances to the next suggestion
4. Press **↑ (ArrowUp)** — highlight moves to the previous suggestion
5. Press **Enter** — the highlighted suggestion is inserted into the field; dropdown closes
6. Type `$` again to reopen suggestions; press **Tab** — same as Enter (inserts suggestion)
7. Type `$` again; press **Escape** — suggestions dismiss without inserting
8. Verify the hint text below the suggestion list reads: **"↑↓ · Enter insert · Esc dismiss"**

## Steps — Node Search overlay keyboard navigation

9. On the canvas, press the **S** key to open the NodeSearchMenu overlay
10. Verify the overlay opens with a search input focused
11. Type a partial node name — results filter dynamically
12. Press **↓** to highlight the first result; **↑** to move up
13. Press **Enter** — the highlighted node is added to the canvas; overlay closes
14. Press **Escape** — overlay closes without adding a node

## Steps — Debug panel preset saving keyboard shortcuts

15. Open the Debug panel; enter a request body and click **Save** (preset flow)
16. When the preset name input is focused, press **Enter** — preset is saved
17. Press **Escape** — preset save is cancelled; input closes

## Steps — standard form keyboard navigation

18. In Config → Variables tab, Tab through the name and value inputs — verify Tab advances focus
19. In Config → API Keys create form, Tab through Label → Scopes → Expiry → submit button
20. Verify Enter in the last field or on a submit button submits the form

## Expected result
- VarField suggestions: ArrowDown/Up navigate, Enter/Tab insert, Escape dismisses
- Node search: ArrowDown/Up navigate, Enter selects, Escape closes
- Debug preset input: Enter saves, Escape cancels
- Config forms: Tab advances focus through fields; Enter submits

## Failure indicators
- ArrowDown key does not move highlight in VarField suggestion list
- Enter in VarField does not insert the highlighted suggestion
- Node search overlay does not respond to arrow keys
- Escape does not dismiss the VarField suggestion dropdown
- Hint text "↑↓ · Enter insert · Esc dismiss" is absent from suggestions

## Severity rationale
Medium: keyboard navigation is essential for power users and accessible workflows; broken keyboard shortcuts force mouse-only interaction.
